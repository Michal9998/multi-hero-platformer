main module

Main module for the Hero Shift game.

This module initializes the Pygame engine, handles the main game loop, manages the state machine (MENU, LEVEL_SELECT, PLAY, SETTINGS, PAUSE, WIN), and orchestrates the rendering and event handling for the entire game.

main.draw_centered_text(surface, text, font, color, y)

Render and draw centered text on a given surface.

Parameters:

  • surface (pygame.Surface) – The surface to draw the text on.

  • text (str) – The string of text to display.

  • font (pygame.font.Font) – The Pygame font object to use.

  • color (tuple) – The RGB color tuple for the text.

  • y (int) – The y-coordinate for the center of the text.

main.load_progress()

Load the player’s level progress from the save file.

Reads the JSON save file to determine the highest unlocked level. If the file does not exist or an error occurs, it defaults to level 1.

Returns:

The highest unlocked level number.

Return type:

int

main.main()

The main game loop and entry point of the application.

Initializes UI elements, handles the game state machine, processes player input, updates game physics, and renders graphics to the screen.

main.save_progress(level)

Save the player’s level progress to the save file.

Writes the newly unlocked level to a JSON file to persist progress between game sessions.

Parameters:

level (int) – The highest level number that the player has unlocked.
