level module

Level parsing and environment management module.

This module handles the loading of Tiled map files (.tmx), processing map layers, extracting collision logic (hazards, deadly blocks, platforms), and rendering the static background environment. It also contains the logic for dynamic terrain, specifically the FragilePlatform.

class level.FragilePlatform(x, y, width, height, image)

Bases: object

A dynamic platform object that breaks a short time after being stepped on.

Once broken, it loses its collision hitbox and disappears, but will respawn after a set duration.

draw(surface, camera_x, camera_y)

Draw the platform on the screen.

If the platform is touched and close to breaking, a visual “screen shake” effect is applied to warn the player.

Parameters:

  • surface (pygame.Surface) – The surface to draw the platform on.

  • camera_x (int) – The current X offset of the camera.

  • camera_y (int) – The current Y offset of the camera.

update(player_rect)

Update the state of the fragile platform.

Checks for collision with the player, manages the countdown to breaking, and handles the automatic respawn sequence.

Parameters:

player_rect (pygame.Rect) – A hitbox representing the area just beneath the player’s feet.

level.draw_level(surface, tmx_data, camera_x, camera_y)

Render static environment tiles onto the game surface.

Iterates over all visible tile layers and draws them, shifting their position based on the camera’s coordinates. Note: The “Fragile” layer is skipped, as those dynamic platforms render themselves.

Parameters:

  • surface (pygame.Surface) – The target surface to draw the tiles on.

  • tmx_data (pytmx.TiledMap) – The loaded map object containing tile logic.

  • camera_x (int) – The X offset for rendering.

  • camera_y (int) – The Y offset for rendering.

level.load_level(filename)

Parse a .tmx map file and extract environmental data for the game engine.

This function reads layers from Tiled map editor files and categorizes tiles into standard platforms, hazards (like spikes), deadly blocks (like acid), and dynamic fragile platforms. It also finds key objects like spawn points.

Parameters:

filename (str) – The file path to the .tmx map asset.

Returns:

A tuple containing: - tmx_data (pytmx.TiledMap): The raw map object (or None if failed). - platforms (list): A list of pygame.Rect objects for standard ground. - hazards (list): A list of pygame.Rect objects for hazards. - fragile_platforms (list): A list of instantiated FragilePlatform objects. - deadly_blocks (list): A list of pygame.Rect objects for instant death areas. - chest_rect (pygame.Rect): The hitbox of the level’s exit/win condition. - spawn_pos (tuple): An (X, Y) coordinate tuple for player spawning.

Return type:

tuple
